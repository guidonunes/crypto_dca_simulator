## üöÄ Features

- **Strategy Comparison:** Simulate and compare **Dollar Cost Averaging (DCA)** vs. **Lump Sum** investing.  
- **Multi-Asset Support:** Built-in support for **Bitcoin (BTC)**, **Ethereum (ETH)**, and **Ripple (XRP)**.  
- **Precision Math:** All financial calculations use `BigDecimal` to avoid floating-point errors.  
- **Persistent History:** Automatically saves simulation results to a **MySQL** database.  

### Design Patterns Implemented
- **Strategy Pattern** ‚Äî easily swap investment algorithms  
- **Factory Pattern** ‚Äî instantiate strategies dynamically  
- **DAO Pattern** ‚Äî clean and testable data persistence
